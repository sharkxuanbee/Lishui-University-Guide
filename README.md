## 前言

在大学学习过程中，课程选取及资源获取是重要环节之一。有时，我们可以轻松地找到所需资源，但有时则需要花费大量时间和精力。因此，这一平台的设立初衷在于提供一个更为便捷的途径，让同学们能够更容易地获取所需的课程资料。

> 本项目受到了很多同类型项目的启发，包括但不限于 [上海交通大学课程分享](https://github.com/kxxwz/SJTU-Courses)，[中国科学技术大学课程资源](https://github.com/USTC-Resource/USTC-Course)，[台州学院电子与信息工程学院（大数据学院）计算机类课程指南]（https://github.com/tzusecu/CS_Courses）

# 1. 章节

## 1.1 水课学习法

对于一些必要但占用过多时间的网络课程，我们推荐使用自动化工具来提高学习效率，从而将更多时间投入到更有价值的学习内容中。

[ocsjs](https://github.com/ocsjs/ocsjs) 是一个开源项目，可以帮助同学们自动化处理网络课程学习任务，解放时间，减少不必要的重复劳动。

对于一些学习通答案 本指导仅供学习，复习参考 不得抄袭 [查看学习通答案文件夹](学习通答案/)
> 注：请合理使用此类工具，确保不违反学校相关规定和课程要求。
本项目不负责任何因使用此类工具而导致的后果，包括但不限于课程不及格、学校处罚等。
